# **App Name**: NurseryNet

## Core Features:

- Preschool Directory: Interactive directory of local preschools with detailed profiles and search functionality.
- Preschool Profiles: Customizable profile pages for preschools, highlighting their unique features and offerings.
- AI Logo Generator & Animator: AI-powered logo creation tool to generate new logos or animate existing logos for preschools, enhancing their branding and making the profile more enticing using Gemini API tool.
- Interactive ROI Calculator: ROI calculator that simulates the benefits of joining NurseryNet for preschool owners, including projected revenue increases and lead generation.
- Profile Preview: Live preview of a preschool's profile to demonstrate what it looks like on the NurseryNet directory from a parent's point of view.
- Preschool Dashboard: Private dashboard for each preschool with analytics on profile views, leads, and engagement, with weekly WhatsApp reports summarizing key data.
- Incentive Offers: Offer incentives for preschool owners to join, such as a discount for the first month or free access to premium features with early signup.

## Style Guidelines:

- Primary color: Medium purple (#9370DB) for a friendly and inviting feel.
- Background color: Light purple (#E6E0FF) with a subtle, grainy gradient texture inspired by Canva, for a soft and modern look.
- Accent color: Soft pink (#FFB6C1) and light blue (#ADD8E6) used sparingly to add pops of color and visual interest without overwhelming the purple scheme.
- Body and headline font: 'Fredoka' (sans-serif) to match the screenshot and create a playful and inviting feel, and complement the softer and slightly rounded feel of Nunito.
- Additional font: 'Nunito' (sans-serif) to match the screenshot, pairing the modern simplicity with the friendly rounded letterforms of Fredoka.
- Flying bird section with purple-blue gradient background, animated 3D bird carrying delivery package, floating stars.
- Colorful horizontal stripes with organic rounded shapes and floating hearts. Yellow background with 3D caterpillar, rainbow pyramid, baby bottle, and floating elements.
- Use custom icons to represent preschool features and services, with a style that complements the 3D elements and animations.
- Implement subtle, playful animations for UI elements like buttons and transitions. Animated 3D elements (bird, caterpillar, pyramid, bottle) to enhance engagement.
